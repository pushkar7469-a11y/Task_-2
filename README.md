# Task_-2
Matrix Operations in C
📌 Project Overview

This project demonstrates the implementation of matrix operations in C using functions and 2D arrays. The program is designed to improve understanding of array manipulation, nested loops, and modular programming by separating each operation into individual functions.

🚀 Features
Accepts matrix input from the user.
Displays matrices in a formatted manner.
Performs basic matrix operations:
Matrix Addition
Matrix Subtraction
Matrix Multiplication
Matrix Transpose
Uses user-defined functions for better code organization.
Supports matrices up to 10 × 10.
🛠 Technologies Used
Language: C
Compiler: GCC / Dev-C++
Concepts: Functions, 2D Arrays, Nested Loops, User Input, Matrix Manipulation
